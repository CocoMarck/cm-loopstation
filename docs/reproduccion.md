# Reproducción
Las opcines de reproducción seran; parar, iniciar, volumen, y silenciar.

El mute/silenciar es de gran importancia para hacer dinamicas con los silencios, si que si. Ademas en vez de parar un pista, para despues iniciarla despues, lo mejor es hacer mute, para mantener dinamicas.

Lo que pasa es que cuando detienes una pista, la reproducción se para abruptamente, y al volver a iniciar, comienza desde el inicio (vaya la redundancia).
